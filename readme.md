# Consumer Goods Domain – Ad-hoc Insights

## Project Overview
This project focuses on solving ten ad-hoc business requests for Atliq Hardwares, a leading computer hardware and consumer goods manufacturer.  
The objective is to extract meaningful insights from the MySQL database and present them to support data-driven decision-making.

## Problem Statement
Atliq Hardwares is expanding rapidly but faces challenges in generating quick insights for business decisions.  
The Data Analytics Director, Tony Sharma, requires immediate analysis for ten ad-hoc requests listed in the document `ad-hoc-requests.pdf`.

The goal of this project is to:
- Query the MySQL database
- Solve the ad-hoc business questions
- Present insights in a format suitable for senior management

## Tools and Technologies
- Database: MySQL  
- Database File: `atliq_hardware_db.sql`  
- Querying Tool: MySQL Workbench  
- Visualization: Power BI  
- Version Control: GitHub

## Approach
1. Imported the SQL dump file into MySQL and reviewed the database structure.
2. Wrote SQL queries to answer all ten ad-hoc requests.
3. Exported query results into Power BI.
4. Built clear, concise visuals to communicate insights effectively.
